# Omok-Pang

## 프로젝트 소개
- 게임 소개 영상 : ["오목팡" 게임은 어떻게 진행될까?](https://youtu.be/K6v6yjD-d7E)
- 오목팡은 **전통적인 오목 규칙에 '특수 카드 시스템'을 더한 전략형 오목 게임**입니다.
- 게임 규칙 : 턴제 방식으로 진행되며, 플레이어는 제한 시간 안에 돌을 두거나 특수 카드를 사용할수 있습니다. 또한, 실시간 멀티플레이를 통한 다양한 매칭(2:2 팀전, 2인/4인 개인전)을 통해 다른 플레이어와 게임을 즐길 수 있습니다.
- 신경쓴 부분 : 특수 카드의 당첨 확률은 각 카드가 승패에 주는 영향을 고려하여, 카드 별로 약 4%에서 20%까지로 구성되게 했습니다. 또한, 직관적인 UI 배치와 명확한 게임 흐름으로 플레이어가 쉽게 게임에 참여할 수 있도록 설계했습니다.
