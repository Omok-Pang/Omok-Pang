## 🗣 프로젝트 소개
오목에 8가지 특수 카드를 더해, 2인용·4인용 개인전과 팀전까지 즐길 수 있는 전략형 대전 게임입니다.

## 🎯 8가지 특수 카드 소개
1. 상대돌 제거(Remove) : 상대가 놓은 돌 1개를 선택해 판에서 제거할 수 있는 공격 카드.

2. 두 번 두기(Double Move) : 한 턴에 두 번 연속 돌을 둘 수 있는 카드.

3. 돌 교환(Swap) : 내 돌과 상대 돌 1개를 서로 교환할 수 있는 카드.

4. 타임락(Time Lock) : 상대의 다음 턴 제한시간을 3초로 줄이는 카드.

5. 한턴 보호(Defense) : 내 다음 턴 동안 상대가 Remove나 Swap을 사용해도 완전히 무효화시키는 방어 카드.

6. 방패(Shield) : Remove·Swap 등 특정 공격을 자동으로 차단하는 보호막 카드.

7. 공용돌(Shared Stone) : 전체 플레이어가 같이 해당 돌을 사용할 수 있는 카드.

8. 폭탄(Bomb!) : 선택한 좌표를 중심으로 주변 돌들을 터뜨려 제거하는 카드.
